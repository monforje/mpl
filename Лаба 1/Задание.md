# Лабораторная работа №1
## Обработка данных CSV с параллельными вычислениями

### Описание задания

Необходимо на Python написать код генерации 5 CSV файлов вида:

**Категория, Значение**

Где:
- **Категория** — случайная буква от A до D
- **Значение** — случайное число с плавающей точкой (float)

Далее нужно **параллельно обработать** эти 5 файлов. Обработка заключается в поиске медианы и стандартного отклонения в рамках каждой категории.

---

## Этапы выполнения

### Этап 1: Генерация исходных данных

Создать 5 CSV файлов: `data_1.csv`, `data_2.csv`, ..., `data_5.csv`

**Требования:**
- 100 строк в каждом файле
- Формат: `Категория, Значение`
- Пример: `A, 23.2`
- Категория — случайная буква от A до D
- Значение — случайное число с плавающей точкой (float)

**Папка:** `1_step/`

---

### Этап 2: Параллельная обработка файлов

Параллельно обработать 5 файлов:

**Задача:** В рамках каждого файла и каждой категории:
- Найти **медиану** значений
- Найти **стандартное отклонение** значений

**Результат для каждого файла:**
```
A, медиана, отклонение
B, медиана, отклонение
C, медиана, отклонение
D, медиана, отклонение
```

**Выходные файлы:** `data_1.1.csv`, `data_2.1.csv`, ..., `data_5.1.csv`

**Папка:** `2_step/`

---

### Этап 3: Создание итогового файла

Прочитать все 5 обработанных файлов и объединить их в один итоговый CSV файл.

**Задача:**
- Для каждой категории найти **медиану медиан**
- Для каждой категории найти **стандартное отклонение отклонений**

**Формат итогового файла:**
```
Категория, Медиана медиан, Отклонение отклонений
A, 25.3, 4.2
B, 30.1, 5.7
C, 28.9, 3.8
D, 26.5, 4.1
```

**Выходной файл:** `data_final.csv`

**Папка:** `3_step/`

---

## Технические требования

- ✅ Вынести формулы расчета медианы и стандартного отклонения в **отдельный модуль**
- ✅ Использовать **pandas** для работы с CSV
- ✅ Использовать **ThreadPoolExecutor** для параллельной обработки
- ✅ Код должен быть простым и понятным (студенческое задание)
